MOTOR DE POVESTE INTERACTIVA - VARIANTA CU PORNIRE SIMPLA
===========================================================

Aceasta arhiva a fost refacuta astfel incat sa nu fie necesar sa configurezi manual proiectele:
- solutia Visual Studio foloseste identificatorul modern pentru proiecte SDK-style;
- este inclus global.json, compatibil cu SDK-ul .NET 9.0.312 instalat pe calculatorul tau;
- sunt incluse scripturi de pornire directa.

PASII RECOMANDATI
-----------------
1. Extrage TOATA arhiva intr-un folder, nu deschide fisierele direct din ZIP.
2. Dublu click pe 00_TESTEAZA_PROIECTUL.bat.
3. Pentru rulare rapida:
   - 01_PORNESTE_PLAYER.bat = aplicatia de citire/rulare;
   - 02_PORNESTE_EDITOR.bat = aplicatia de editare.
4. Pentru cod in Visual Studio: 03_DESCHIDE_IN_VISUAL_STUDIO.bat.

TEST RAPID PLAYER
-----------------
Dupa pornirea Player-ului:
Fisier > Deschide poveste... > selecteaza samples\MareaEvadare.zip

CERINTE
-------
- Windows 10/11
- .NET SDK instalat (SDK-ul tau 9.0.312 este acceptat)
- Pentru deschiderea in Visual Studio: workload-ul .NET desktop development

Daca un script afiseaza o eroare, captura necesara este din acea fereastra,
nu dintr-un proiect gol sau din folderul gresit.
